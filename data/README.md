# Data

Full dataset is not committed here. Source: Islam et al. (2020) Prothom Alo
Bengali sentiment corpus, 17,852 rows, 3-class (Neutral/Positive/Negative).

On Kaggle, mounted at:
```
/kaggle/input/datasets/reversedthoutgts/bangla-dataset/train_.csv
/kaggle/input/datasets/reversedthoutgts/bangla-dataset/test_.csv
```

Locally (for development against the same data), see `../Dataset/train.csv`
and `../Dataset/test.csv` in the project root.

Columns: `Data` (Bengali text), `Sentiment` (0=Neutral, 1=Positive, 2=Negative).
